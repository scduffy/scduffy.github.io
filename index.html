In this tutorial I will go through and analyze vehicle crashes in maryland statewide and see if there are any interesting patterns. I will also investigate how road conditions affect crash numbers as that has always been interesting to me, ie seeing how wet or icy roads increases the number of crashes.

# plot a given dictionary as a scatter plot. Options for adding in polyfit line of specified degree, if 0 don't plot a polyfit line.
def plot_dict(dict_in, title, xlab, ylab, degree, height, width):
    # prepare to plot the dict
    lst = dict_in.items()
    lst = sorted(lst)

    # seperate the key's and value's into two seperate arrays corrosponding to x-axis and y-axis of plot
    x, y = zip(*lst)

    # plot it with labels and title
    plt.figure(figsize=(width, height))
    plt.title(title)
    plt.xlabel(xlab)
    plt.ylabel(ylab)
    plt.scatter(x, y)

    # optionally add a polyfit line if degree is greater than 0
    if degree > 0:
        poly_fit_model = np.poly1d(np.polyfit(x,y,degree))
        poly_line = np.linspace(x[0], x[-1], 50)
        plt.plot(poly_line, poly_fit_model(poly_line), color='red')

Daily Averages¶

First thing I want to look at is the average number of accidents per day between 2015 and early 2020. Using the average per day might give us a better understanding of how accident numbers trend. It will also shrink our x axis down from 5 years to 1 year, which should give us a higher resolution on our data since there can only be 365 points vs the 100,000 we could have had before. One thing to not about the plot is that instead of being actual dates like 01/01/2020 in the above plots, since we are averaging the same day over multiple years it is easier to represent the x axis as "day of year" since they would be the same over multiple years. This is not as complicated as you might expect since the day of year is just an integer between 1 and 364 with january 1st being 1 and december 31st being 365. Also, since the spike in accidents in mid 2020 was odd considering the rest of the data, we will not consider it when calculating daily averages as it would clearly skew the data.

Hypothesis Testing¶

Now that we have a general idea about our data, lets look at some hypothesis about it and try to draw some deeper conclusions. We will do one hypothesis test for each of the upper and lower clusters and one final two sample hypothesis test comparing the two clusters. For consistency, we will use an alpha (p-value cutoff) of .05 for all of our hypothesis tests.

Our null (H0) and alternative (HA) hypothesis' for each of our hypothesis tests will be the same: is the mean difference of average number of accidents per day 0.

Hypothesis Test Function¶

Lets just make a function to do this since it is the same code repeated. To calculate this, we will need to first get difference data for each cluster. In this case that means getting an array containing the difference between the number of accidents on a given day and the mean number of accidents over the entire cluster. We calculate this by first taking the mean using statistics.mean for ease, then using a for loop iterating and getting the difference while appending the result to an array. We need to do this for every cluster and that is done below.

Now that we have difference data, we can hypothesis test. We will define H0: mu = 0 and HA: mu != 0 as our null and alternative hypothesis' respectively.

In [89]:
def differences(data):
    # get differences 
    diffs = []
    xbar = statistics.mean(data.values())
    for day in data: diffs.append(data[day] - xbar)
    return diffs

# data: dictionary with datetime key and integer value
def hypothesis_test(data):
    # get differences 
    diffs = differences(data)
    # get mean, std, and sqrtn for z-score calculation
    mu = 0
    xbar = statistics.mean(diffs)
    s = statistics.stdev(diffs)
    sqrtn = math.sqrt(len(diffs))

    # do the test
    zscore = (xbar-mu)/(s/sqrtn)

    print("xbar: ", xbar)
    print("stdev: ", s)
    print("z-score: ", zscore)
    print("p-value: ", end='')

    # get the p value (two tailed so multiply by 2)
    return scipy.stats.norm.sf(abs(zscore))*2

Two Sample Z Test¶

The previous hypothesis tests were not that interesting and testing to see if the difference between the two means is close to zero is redundant as we have already concluded that for each one individually. Instead, lets try the two sample test looking at the means of the upper and lower clusters directly and not the mean differences. For this test we will set H0: mu1 - mu2 == 0, and HA: mu1 - mu2 != 0.

After the calculation, we can see that the z-score is much further from zero compared to the previous ones, with the absolute value being around 67 and the absolute values of the previous two being very close to zero. This means for us that we reject that the true means of the upper and lower cluster are equal in favor of the alternative that they are different, which makes sense. Based on the graphs we plotted earlier the upper and lower clusters were very different and that is why we seperated them to look at individually in the first place. So it makes sense that we can confirm with a hypothesis test that the means are different.

Insights Gained¶

So what have we learned from all of this? The first and most notable thing that we learned early on is that something happened in early 2020 that caused there to be an great increase in the daily number of accidents. It is likely due to COVID-19 that this occured but things like vaccination or infection status of those involved in accidents is not reflected directly in our dataset. So this is only speculative.

The other thing we learned from analyzing the spike in 2020 seperately from the rest of the data is that the road conditions were not responsible for the increase in accidents as the distribution of accidents and the road conditions that they occured in matched the rest of the data. We also concluded from our data that wet or snowy road conditions were also not responsible for accidents as the greatest number of accidents occured in 'Dry' road conditions. This could likely be analyzed further by looking into specific months that had wet or snowy road conditions and looking at the proportion of dry to wet/snowy road conditions in those months but that was not a part of this tutorial.

Lastly we concluded that within each cluster, the average difference between the number of accidents for any given day and the mean number of accidents for the cluster overall was 0. At the same time we also concluded that the means of the two clusters was completely different, which was quite interesting. There is a lot of information in this dataset that has not been analyzed in this short tutorial and many more insights can be gained if you wish to follow this tutorial for something else in the future.
